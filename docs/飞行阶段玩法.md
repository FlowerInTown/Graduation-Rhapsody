# 宇宙电台
## UI和交互
在界面的下方，是工具栏UI界面，显示一个电台的按钮，使用资源：assets\ui\radio.png
点击需要有个晃动的效果
点击后开始播放宇宙电台的音效和文本
## 文本数据
数据放在src/data/radio-data.js中
随机播放其中一句话的文本。
文本显示在屏幕中间偏上的地方
## 语音数据
随机播放一条语音，播放结束后，切换下一条文本
语音资源在目录：assets\music\radio


# CD机玩法
## UI和交互
在界面的下方，是工具栏UI界面，显示一个CD机的按钮，使用资源：assets\ui\cdji.png
点击需要有个晃动的效果
点击后在画面中间生成抽屉界面：assets\ui\drawer.png
一行三个，最多三行显示9个cd，实际是显示已玩家经获取的cd列表，点击一个cd按钮，开始播放这个cd对应的音乐
cd按钮鼠标悬浮也会有这个cd的描述（对应cd数据的hint字段）
