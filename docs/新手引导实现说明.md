# 新手引导实现说明

## 功能概述
已实现初始关卡的新手引导流程，玩家第一次到达起始行星时会自动触发。

## 实现内容

### 1. 新建文件
- `src/systems/tutorial.js` - 新手引导系统

### 2. 修改文件
- `src/data/game-state.js` - 添加 `tutorialCompleted`、`tutorialEnergyHarvested`、`tutorialFinalShown` 标记
- `src/scenes/planet-explore.js` - 集成新手引导系统
- `src/main.js` - 添加探测器音效加载

## 功能特性

### 引导流程

#### 第一阶段：基础引导（3句台词）
1. **三句固定台词**，依次显示：
   - "生命探测器看起来没有反应，这里又没有旅行者吗？"
   - "有旅行者就好啦，有的话就可以奏响这个家伙和旅行者打个招呼"
   - "还是先补充一下飞船的能量吧..."

2. **每句台词持续3秒**，不可跳过

3. **箭头指示**：
   - 第一句 → 指向生命探测按钮
   - 第二句 → 指向召唤朋友按钮
   - 第三句 → 指向收集能量按钮

4. **引导期间限制**：
   - 禁用所有按钮点击
   - 隐藏按钮悬浮提示
   - 隐藏星系地图按钮
   - 玩家无法操作，只能观看引导

#### 第二阶段：能量采集
5. **引导完成后**：
   - 按钮恢复可点击状态
   - 玩家可以采集能量
   - 星系地图按钮仍然隐藏

#### 第三阶段：最终引导
6. **采集能量后**：
   - 星系地图按钮显示
   - 触发最终引导台词："让我打开银河地图看看下个目的地去哪里"
   - 箭头指向星系地图按钮
   - 持续3秒后结束

7. **完成后**：
   - 设置 `gameState.tutorialCompleted = true`
   - 之后再访问该行星不会触发引导
   - 玩家可以正常操作所有功能

### 触发条件
- 仅在起始行星（startSystemId 的第0颗行星）
- 仅首次访问时触发
- 降落完成后自动开始

### 音效
- 点击探测器按钮时播放 `detector.wav` 音效

### 对话展示
- 使用角色对话气泡显示引导台词
- 气泡位于角色头像右上方
- 支持自动换行

## 技术细节

### 箭头显示
- 使用 `assets/ui/index.png` 作为箭头图标
- 箭头位于按钮左上方
- 带有脉冲动画效果（透明度0.6-1.0）

### 对话气泡样式
- 使用 `ui_bubble_left.png` 气泡图片
- 棕色文字，居中显示
- 支持多行文字自动换行

## 配置参数

在 `src/systems/tutorial.js` 中可以调整：
- `TUTORIAL_DURATION_PER_LINE = 3` - 每句台词显示时长（秒）
- `ARROW_SCALE = 0.05` - 箭头缩放比例
- `ARROW_OFFSET_X = -20` - 箭头X轴偏移
- `ARROW_OFFSET_Y = -20` - 箭头Y轴偏移

## 测试方法

1. 启动游戏
2. 开始新游戏
3. 飞船降落到起始行星后，会自动触发新手引导
4. 观察三句台词依次显示，箭头指向对应按钮
5. 9秒后引导结束，按钮可点击
6. 点击采集能量按钮
7. 采集完成后，星系地图按钮显示
8. 自动触发最终引导台词，箭头指向星系地图

## 注意事项

- 新手引导只在第一次访问起始行星时触发
- 如果重置游戏状态，引导会再次触发
- 第一阶段引导期间玩家无法进行任何操作
- 必须完成能量采集才能看到星系地图按钮
- 引导完成后会自动解除所有限制
