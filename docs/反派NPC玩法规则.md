# 刷新机制
正常NPC不刷的行星，有一定概率刷新反派NPC（概率可以配置），资源路径: assets\characters\bad_npc
随机选个这个目录下的资源

# 流程说明
1. 不用触发生命检测按钮（下方四个按钮都隐藏），等角色播完到达新行星的固定台词后，反派NPC自动出现，出现后玩家切换资源：assets\characters\astronaut\fear.png
2. 然后开始如下对话
    配置在bad-npc-lines.js
3. 播完这段对话后，系统提示【你的飞船能量被抢走了一半】
4. 飞船能量扣除一半，向下取整
5. NPC说：【谢谢你的帮助哦，嘻嘻嘻嘻...】
6. NPC离开屏幕
7. 主角说【终于走了...】，然后恢复正常玩家图片资源
8. 恢复下方四个按钮显示
