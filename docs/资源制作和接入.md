  命名规则

  - 全小写，单词间用下划线 _
  - 行星背景固定 bg_0.png ~ bg_3.png
  - NPC 文件名 = NPC name（moonlie.png）
  - 音乐文件名 = musicName 字段值（moonlie_song.mp3）
  - 不要用中文、空格、特殊字符

  接入方式

  放进对应目录后，代码里通过 assets.load(key, 'assets/xxx/yyy.png') 加载，音乐通过
  musicPlayer.loadTrack('moonlie_song', 'assets/music/moonlie_song.mp3') 加载。没有对应文件时自动 fallback
  到当前的程序化渲染/合成音，所以你可以一个一个替换，不用一次全做完。

  构建时 node build.js 会自动把 assets/ 下所有图片和音频 base64 内联到 dist/game.html。
