# 触发条件
进入【故乡】这个行星探索场景的时候立即触发


# 流程说明
## 初始化
不需要执行正常的行星探索逻辑：角色的固定触发文本、生命探测器功能等
需要直接屏蔽下方的所有按钮
但是BGM正常播放

## NPC出场
根据npc-data中的with_ending字段，也就是当前所有遇到的npc的实例数据，根据with_ending来判断是否加入这个出场顺序
按照顺序出场：根据npc-data中配置的ending_dialogue的内容来逐个npc对话，npc先说第一句台词
说完后，npc向右离开屏幕，换下一个npc

## 播放结尾流程
NPC所有都对话完成离场后，开始播放结局的BGM，在目录：assets\ui\happyending下，按照 1 2 3 的顺序循环播放目录下的所有音乐，音乐都用数字命名了
播音乐的同时播放结局的图片资源，结局的图片资源放到了assets\ui\happyending目录中，根据这里出场的NPC来选择要播放哪些图片，图片名称我都改成了npc的名称
每个图片的存在时间大概4s，可以配置，图片之间切换要有显隐


## 播放开发者信息流程
开始播放开发者信息，和燃料耗尽结局一样
