# 行星探索的UI设计
1. 左下角显示玩家的头像
2. 右上角显示银河系/当前星系 地图入口按钮
3. 下方显示一排操作按钮，目前只做一个【采集氦3】按钮，点击会触发一个读条，读条结束后，回复飞船能量值，假设星球氦3储藏量为N，回复的值为V=300*N

# 飞行画面的UI设计
1. 右上角显示银河系/当前星系 地图入口按钮
2. 左上角显示音乐列表入口按钮
    点击这个按钮，展开一个列表，用来显示当前角色获取的音乐库
    点击任意一条音乐开始播放当前音乐
3. 加速飞行按钮
    位于游戏画面的下方，点击后，当前飞船航速变为1.5倍，再点一次变为1倍
    高航速状态，按钮显示状态1，普通航速显示状态2

# 银河系地图
玩家当前所在的星系使用闪烁特效标记，点击任意星系唤出【当前星系的地图】，点左上角返回按钮可以返回【银河系地图】
