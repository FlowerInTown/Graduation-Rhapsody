# 初始关卡
游戏开始后来到的第一个行星场景，临时设置为初始关卡
星系地图按钮暂时隐藏

# 新手引导流程
角色播完固定文本后，开始进入新手引导流程，设计如下
角色开始说新手引导的台词
["生命探测器看起来没有反应，这里又没有旅行者吗？", "有旅行者就好啦，有的话就可以奏响这个家伙和旅行者打个招呼", "还是先补充一下飞船的能量吧..."]
上面三句台词分别固定存在2s，不可以跳过
每句台词生效期间，箭头（assets\ui\index.png）指向下面三个按钮的左上方，依次指：生命探测按钮、召唤朋友按钮、收集能量按钮
这三个按钮也要不可点击

引导台词播完后，按钮恢复点击，完成能量采集后，星系地图按钮恢复显示
角色说一句台词：“让我打开银河地图看看下个目的地去哪里”，箭头指向银河地图按钮（注意这个箭头图标是）


# 离开初始关卡
离开后，临时初始关卡就不生效了，这个行星可以再次返回，不再触发新手引导流程
