# 美术资源规格说明 / Art Asset Specifications

## 基本信息
- 游戏分辨率: **1280 × 720** (16:9)
- 底部物品栏高度: **80px** (实际游戏画面区域 1280×640)
- 图片格式: **PNG** (透明背景) 或 **JPG** (无透明需求的背景)
- Sprite Sheet: 水平排列，每帧等宽

---

## 美术风格参考
- **色调**: 深色太空背景 + 明亮发光元素
- **风格**: 类似银河历险记 (Samorost) — 梦幻、有机、充满细节
- **特效**: 场景中有大量发光粒子，物体会有柔和的辉光
- **锚点**: 所有独立对象图片的锚点在中心（代码中以中心坐标绘制）

## 工作流程
1. 制作好图片放到对应目录
2. 用本地 HTTP 服务器运行 `index.html` 测试（`npm run dev`）
3. 图片会自动替换对应的程序化绘制 — 无需改代码
4. 测试满意后运行 `npm run build` 打包成单文件
