# 游戏主流程说明
1. 游戏启动后显示主菜单, 参考/docs/菜单说明.md
2. 点击开始游戏后进入游戏的主逻辑，首先进行银河系信息生成，信息生成逻辑如下
    2.1 生成银河系中的15-30个随机位置的星系位置
    2.2 随机找一个星系位置作为玩家的出发点，以及银河系另一端的某个区域，作为玩家的目的地，故乡行星所在星系（不一定要完全对称）
    2.3 生成每个星系的详细数据，逻辑如下
        2.3.1  每个星系只有一个恒星，恒星的生成概率为：红矮星（最常见，约 70%–80%）、类太阳恒星（约 10%–20%）、红巨星（5%–10%）、白矮星（约 5%–10%，恒星尸体）
        2.3.2  每个恒星的大小分为五个挡位：60%标准大小、80%标准大小、标准大小、150%标准大小、200%标准大小。电脑横屏模式下，标准大小约占到屏幕高度的五分之一。
        2.3.3 每个恒星的周围随机生成 1-3 个行星，每个行星数据生成参考 【单个星球设计实现框架.md】
3. 生成星系信息后，

5. 在星系的地图UI中点击一个星球，弹框提示：是否要前往该星球，点击【是】，进入【飞船飞行画面】，点击【否】弹框消失。进入飞行时需要计算本次飞行消耗：逻辑如下：
    5.1 出发行星和目的行星是否是同一个星系的，是的话，固定扣除【星系内飞行消耗】来减少飞船的能量值，具体参考【数值设计.md】
    5.2 如果不是一个星系的行星，则按照星系之间的距离来计算飞行消耗 具体参考【数值设计.md】
6. 在飞船飞行画面，可以点击【跳过按钮】直接结束当前流程，如果没有点，则在保底时间（1分钟）到达后自动结束。飞行画面结束后，进入行星探索阶段。
7. 行星探索阶段实现参考【单个星球设计实现框架.md】
8. 点击降落在行星上的飞船 唤出 【当前星系的地图】，点左上角返回按钮可以返回【银河系地图】，以便玩家再次启程出发
9. 游戏结束条件1：在飞船飞行画面检测飞船能量是否<=0，是的话进入【游戏结束流程】
10. 游戏结束条件2：玩家回到故乡行星。先进入【回到故乡剧情】，再进入【游戏结束流程】


# 飞船飞行画面
1. 角色驾驶的太空船在屏幕中间偏左一点点，背景是逐渐移动的星空，飞船航行速度为1倍的时候，星空缓慢移动，飞船航行速度为1.5倍的时候，星空加速移动对应倍率
2. 播放随机的BGM，氛围感的重要部分
3. 进入飞行画面后立即进行飞行消耗扣除，如果满足游戏结束条件1（也就是飞船能量<=0），禁止玩家再打开星系地图，进入【游戏结束流程】
4. 飞行画面存在的时间根据当前航行距离来计算，假设计算出来的距离是500，耗时为 T=500*0.02 （单位：秒）


# 游戏结束流程
1. 如果是在飞船飞行画面中，则放置一层半透明遮罩，从下往上滚动一些字幕信息，使用【游戏结束文本.md】的文本，禁止玩家再打开星系地图
2. 如果是触发了【回到故乡剧情】，则等到【回到故乡剧情】播放完成后，再放置一层半透明遮罩，从下往上滚动一些字幕信息，使用【游戏结束文本.md】的文本，禁止玩家再打开星系地图
3. 字幕播放完毕后，弹出【回到主菜单】按钮，点击回到游戏启动时的主菜单
